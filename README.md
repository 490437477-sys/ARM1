# 舵机机械臂控制系统使用手册

## 1. 产品概述

本产品是一款 **4舵机机械臂控制系统**，支持三种控制方式：

| 控制方式 | 说明 |
|---------|------|
| 摇杆控制 | 双摇杆实时控制 |
| 串口命令 | Arduino IDE 串口监视器 |
| Python界面 | 电脑可视化控制 |

## 2. 硬件配置

- Arduino UNO R3 (主控制器)
- 2个双轴摇杆模块
- 4个MG90S舵机

## 3. 引脚定义

### 舵机引脚

| 舵机 | 引脚 |
|------|------|
| 舵机0 | D5 |
| 舵机1 | D9 |
| 舵机2 | D10 |
| 舵机3 | D11 |

### 摇杆引脚

| 摇杆 | X轴 | Y轴 | 按钮 |
|------|-----|-----|------|
| 摇杆1 | A0 | A1 | D2 |
| 摇杆2 | A3 | A2 | D4 |

## 4. 控制逻辑

| 输入 | 控制对象 | 角度范围 |
|------|---------|---------|
| 摇杆1 X轴 | 舵机0 | 0-90° |
| 摇杆1 Y轴 | 舵机1 | 0-90° |
| 摇杆1按钮 | - | - |
| 摇杆2 X轴 | 舵机2 | 0-90° |
| 摇杆2 Y轴 | 舵机3 | 0-180° |
| 摇杆2按钮 | - | - |

## 5. Arduino设置

1. 在Arduino IDE中打开 `servo_control_zh.ino`
2. 选择开发板: **Arduino UNO**
3. 选择正确的串口
4. 上传代码
5. 打开串口监视器（波特率9600）

## 6. 串口命令

### 舵机控制命令

| 命令 | 功能 |
|------|------|------|
| `0 90` | 设置舵机0为90度 | 
| `1 45` | 设置舵机1为45度 | 
| `2 90` | 设置舵机2为90度 |
| `3 90` | 设置舵机3为90度 |

### 快捷命令

| 命令 | 功能 |
|------|------|
| `4 45` | 设置舵机4为45度 |
| `z` | 所有舵机初始化 |
| `a` |  查看所有角度 |
| `h` | 显示帮助 |

## 7. Python界面

### 环境要求

- Python 3.x
- pyserial库

### 安装与运行

```bash
pip install pyserial
python servo_control.py
```

### 界面功能

- 端口选择与刷新
- 连接/断开控制
- 滑块控制角度
- 数字输入控制
- 归零按钮

## 8. 安全警告

### 电源要求

上传代码后，请将电源从电脑USB切换到 **外部电源(9V 1A或更高)**。

### 过载保护

- 过载夹取物体时请勿施加过大力量
- 舵机堵转会导致电流激增并可能使开发板重启
